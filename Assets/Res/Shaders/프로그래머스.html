프로그래밍
다시 풀 문제
문제 설명

아래 그림처럼 1 x 1 크기 정사각 격자로 이루어진 n x n 크기의 게임 맵이 주어질 때, 공이 맵 밖으로 나가기 전까지 움직인 거리를 구하려고 합니다.

  • 공은 맵의 맨 왼쪽 위 칸의 왼쪽 면 중앙에서 출발하여 오른쪽 방향으로 이동합니다.
  • 맵의 각 칸은 빈칸이거나 5가지 장애물(정사각형 1가지, 직각삼각형 4가지)중 하나가 놓여있습니다.
그림
장애물 번호 1 2 3 4 5
  • 공은 장애물에 닿으면 움직이는 방향이 바뀝니다.
  • 각 장애물에는 내구도가 있습니다. 장애물에 공이 닿으면 장애물의 내구도가 1만큼 줄어듭니다. 내구도가 0이 되면 장애물은 사라집니다.
    • 장애물의 내구도가 1인 상태에서 공이 닿을 경우 공의 방향이 달라진 후 장애물은 사라집니다.
    • 왼쪽 그림은 공이 2번 → 4번 → 3번 → 1번 → 5번 장애물에 부딪히는 과정을 나타냅니다. 공이 모든 장애물에 한 번씩 닿았으므로 모든 장애물의 내구도가 1씩 감소합니다.
    • 오른쪽 그림은 공이 5번 장애물에 부딪힌 이후 맵 밖으로 나가기까지 이동하는 과정을 나타냅니다. 공이 1번, 3번 장애물에 한 번씩 닿았으므로 1번, 3번 장애물의 내구도가 1씩 감소합니다.

공이 움직인 거리는 12(빨간 화살표의 길이 합)입니다.

게임 맵의 장애물 정보를 담은 2차원 정수 배열 board와 각 장애물의 내구도를 담은 2차원 정수 배열 durability가 매개변수로 주어질 때, 공이 맵 밖으로 나가기 전까지 움직인 거리를 return 하는 solution 함수를 완성해 주세요. 단, 처음부터 공이 맵 안으로 들어갈 수 없는 경우에는 0을 return 해주세요.

제한사항
  • 3 ≤ board의 행 길이 = board의 열 길이 = durability의 행 길이 = durability의 열 길이 ≤ 10
  • board의 원소는 0, 1, 2, 3, 4, 5 중 하나입니다.
    • 0은 장애물이 없는 칸을 의미합니다.
    • 1, 2, 3, 4, 5는 위에서 설명한 장애물 번호를 의미합니다.
  • durability[i][j]는 board[i][j]의 장애물의 내구도를 의미합니다.
    • 0 ≤ durability의 원소 ≤ 10
    • board[i][j]가 0일 경우(장애물이 없는 칸)에만 durability[i][j]는 0입니다.
